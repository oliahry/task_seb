# task_seb
Task for Junior Quantitative Analyst in Credit Risk Model Validation done by Olha Hrytsenko

The task was completely executed in GoogleColab web. 

Before starting the code, please execute the following steps to ensure that the project is replicable.

Steps
1. Go to GoogleColab by following the link https://colab.research.google.com/ or manually searching for GoogleColab in the web. You will need to login to your Google account to use the page (viewing works without login)
2. File -> Open Notebook -> Select GitHub option -> Enter "oliahry" in the search -> Select "oliahry/task_seb" -> Select "code_and_report.ipynb"
3. Wait for the page to connect to the environment. You can see if it is connected in the upper right corner. 
4. Upload csv files provided in the repository to the GoogleColab local environment.
5. Run the code

Or 

1. Download all csv files from repostitory.
2. Download code_and_report.ipynb file from the repository.
3. Open GoogleColab.
4. File -> Open Notebook -> Upload -> Select "code_and_report.ipynb" from your local device.
5. Add all csv files to GoogleColab environment.
6. Run the code
